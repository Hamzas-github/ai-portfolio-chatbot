## How does this chatbot work?
It's a project of his own. A small framework-free widget backed by a server that holds the API keys. The AWS-native version answers from Amazon Bedrock with a RAG Knowledge Base over his CV/projects notes (Amazon S3 Vectors + Titan v2 embeddings, behind a Lambda), and ElevenLabs reads each answer aloud in a clone of his voice. The original version used a Cloudflare Worker with Groq for the words.
